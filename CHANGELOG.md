# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-08
- Initial public SDK release.
- Added `OAuthClient` with authorize URL builder, callback parsing, token exchange, and profile resolution.
- Added cURL-based HTTP client and typed response model.
- Added compatibility parser for malformed callback query format.
- Added Composer autoload support.
- Added non-Composer bootstrap loader (`bootstrap.php`).
- Added runnable examples and installation guide.
